pyRefine.batch

A public utility to ingest, characterize, and filter large datasets for the refinement of Volumetric Efficiency tables.


pyRefine batch execution API

(1) Upload a data log file for consumption and analysis, (2) the volumetric efficiency table corresponding to the data log file, (3) the associated MAP index, and (4) the RPM header, then (5) execute the 'refine-batch-execution' endpoint.
